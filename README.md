# MediaSync
Comparing media file lists in nested directories
```
usage: mediasync.py [-h] (-g GENERATE | -c COMPARE COMPARE)

Generating and comparing multimedia file lists

optional arguments:
  -h, --help            show this help message and exit
  -g GENERATE, --generate GENERATE
                        Lists all leaf directories under a specific root
                        directory and saves a file.
  -c COMPARE COMPARE, --compare COMPARE COMPARE
                        Takes two multimedia list paths and compares them.
```
